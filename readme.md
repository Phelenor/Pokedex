## Tech Stack 

- Written 100% in Kotlin
- Coroutines + LiveData
- Dagger + Hilt
- SQLite + Room
- Retrofit2 & OkHttp
- MVVM Architecture
- Paging3
- Kotlin Coil
- Material Design
- [ProgressView](https://github.com/skydoves/progressview)
